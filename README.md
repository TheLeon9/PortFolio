## 🦾 Getting Started

1.  ⚔️ **Installation:**
    Clone this repository locally with `git clone`.

```
git clone [url here]
```

2. 🛡️ **Dependencies:**
   Install dependencies with `npm install` or `npm i`.

```
npm i
```

3. 🏹 **Launch:**
   Start the project with `npm start` or `npm run dev`.

```
npm run dev
```
